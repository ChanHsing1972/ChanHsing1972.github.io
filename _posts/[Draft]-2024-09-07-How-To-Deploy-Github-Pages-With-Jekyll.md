## 0 - 简介

原理：利用基于 [Jekyll](https://jekyllrb.com/) 制作的模板，对内容进行自定义，最终托管到 Github Pages，形成个人博客。

优势：能够方便快捷地搭建起个人博客，无需费用，无需专业知识，技术门槛较低。

笔者使用的模板为 [Chirpy](https://chirpy.cotes.page/)，故本文将从该模板的角度出发进行介绍。更多模板[请见此处](http://jekyllthemes.org/)。
